# Topic Modeling
Topic Modeling using Latent Dirichlet Allocation and Non Negative Matrix Factorization

## Required:
* Python 3.5
* Sklearn

## Execution:
* python topicmodeling.py

* data file (20 Newsgroup Dataset)
